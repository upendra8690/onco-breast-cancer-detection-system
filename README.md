# 🚀 ONCO Breast Cancer Detection System (2026)

## 🌟 Overview
This is an AI-powered healthcare system developed to assist in early detection of breast cancer using deep learning.

## 💡 Key Features
- AI-based breast cancer prediction
- Emergency hospital locator
- Patient management system
- Real-time analytics dashboard

## 🛠 Technology Used
- Python
- Machine Learning (CNN, VGG19)
- Streamlit
- SQLite Database

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/upendra8690/onco-breast-cancer-detection-system.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   streamlit run app.py
